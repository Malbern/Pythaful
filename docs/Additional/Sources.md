This app is for developers.\n\nUTI Inspector shows the Unitform Type Identifier (UTI) information about any document on your device: which UTIs are registered for a given file extension, which apps declared them and how they did it.\n\nWhen you create a document-based app, it is sometimes helpful to know UTIs declared by third-party apps. Firstly, they can be useful as an example of correct UTI declaration. Secondly, they help to make your app compatible with third-party file formats (you can see which UTIs to import in your app). Finally, UTI Inspector helps in debugging any issues related to conflicting UTI declarations
